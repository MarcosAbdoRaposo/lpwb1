1. Declaração com let:

Crie uma variável let idade e atribua sua idade.
Depois, atribua um novo valor representando a idade daqui a 5 anos.
Mostre os dois valores no console: “Minha idade hoje é e daqui a 5 anos será ”

2. Uso de Const:

a. Declare uma constante PI com valor 3.14159. Tente atribuir outro valor a PI e veja o erro no console. Explique por que isso acontece.
b. Declare uma constante SemanasDoAno sem atribuir valor. Na linha seguinte atribua o valor 52. O que ocorre?

Resposta a: uma variável definida como Const não pode ter o valor alterado.

Resposta b: não é possível declarar uma variável com const sem atribuir o valor.
